# Brain-Computer Interface Virtual Arm

**Goal:** Decode motor imagery EEG signals to control a virtual arm in real time.

**Status:** In Progress

**Skills Demonstrated:**  
- EEG preprocessing, artifact handling, spectral/time-frequency feature extraction  
- CNN-based decoding of motor imagery  
- Simulation and mapping to a virtual arm  
- Planning for closed-loop adaptive control  

**Planned Pipeline:**  
1. Preprocess EEG signals (filtering, artifact removal)  
2. Extract spectral features  
3. Train CNN classifier  
4. Map predictions to virtual arm simulation  
5. Evaluate performance (cross-validation, feature importance)  

**Dependencies:** Python, NumPy, PyTorch, Matplotlib, PyBullet  

*This project reflects skills and methodology outlined in my CV: neural decoding, modeling, and computational neuroscience research.*
