# Brain-Computer Interface Virtual Arm

**Goal:** Decode motor imagery EEG signals to control a virtual arm in real-time.

**Status:** In Progress

**Planned Pipeline:**
1. Preprocess EEG signals (filtering, artifact removal)
2. Extract spectral features
3. Train CNN classifier
4. Map predictions to virtual arm simulation
5. Evaluate with cross-validation and feature analysis

**Dependencies:** Python, NumPy, PyTorch, Matplotlib, PyBullet
