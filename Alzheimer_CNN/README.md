# MRI-Based Alzheimer’s Disease Classification Using CNN

**Goal:** Detect early-stage Alzheimer’s disease from structural MRI scans using convolutional neural networks.

**Status:** In Progress

**Skills Demonstrated:**  
- MRI data preprocessing (normalization, resizing, data augmentation)  
- CNN model development for image classification  
- Cross-validation, accuracy, precision/recall, and F1-score evaluation  
- Feature visualization and interpretability analysis  
- Scalable pipeline design for future multimodal neuroimaging studies  

**Planned Pipeline:**  
1. Preprocess MRI images  
2. Train CNN models  
3. Evaluate with cross-validation and performance metrics  
4. Visualize feature maps and analyze model decisions  
5. Prepare pipeline for scaling to additional datasets  

**Dependencies:** Python, TensorFlow/PyTorch, NumPy, Matplotlib, Nibabel  

*This project reflects skills and methodology outlined in my CV: neuroimaging analysis, computational neuroscience, and deep learning applications.*
