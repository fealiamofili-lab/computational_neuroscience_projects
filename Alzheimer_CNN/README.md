# Alzheimer’s MRI Classification

**Goal:** Detect early-stage Alzheimer’s disease from MRI scans using CNNs.

**Status:** In Progress

**Planned Pipeline:**
1. Preprocess MRI images
2. Split into training/testing sets
3. Build CNN model
4. Train and validate
5. Evaluate with accuracy metrics and visualizations

**Dependencies:** Python, TensorFlow/PyTorch, Nibabel, NumPy, Matplotlib
