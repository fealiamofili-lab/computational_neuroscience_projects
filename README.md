# Computational Neuroscience Projects

This repository showcases in-progress projects in computational neuroscience, neural decoding, and biologically inspired learning. Each project demonstrates structured research pipelines, methodology, and active development aligned with the skills listed in my CV.

## Projects

### 1. Alzheimer’s CNN Classification
- **Goal:** Detect early-stage Alzheimer’s disease from MRI scans using convolutional neural networks.  
- **Status:** In Progress (implemented CNN, Grad-CAM interpretability, and RAG clinical knowledge integration).  
- **Highlights:**  
  - Preprocessing pipelines and ResNet-18-based CNN for binary classification (Healthy vs Alzheimer’s).  
  - Grad-CAM heatmaps for model explainability.  
  - Framework-free RAG pipeline for integrating clinical knowledge.  
- [Project folder](Alzheimer_CNN/)

### 2. BCI Virtual Arm
- **Goal:** Decode motor imagery EEG signals to control a virtual arm in real time.  
- **Status:** In Progress  
- **Highlights:**  
  - EEG preprocessing (band-pass filtering, artifact handling) and feature extraction.  
  - CNN models for decoding imagined hand movements mapped to a virtual arm.  
  - Evaluation using cross-validation, confusion matrices, and feature importance analysis.  
- [Project folder](BCI_Virtual_Arm/)

### 3. Spiking Neural Network Simulation
- **Goal:** Model cortical microcircuits using excitatory/inhibitory neurons with STDP learning rules.  
- **Status:** In Progress  
- **Highlights:**  
  - Network simulations using Brian2.  
  - Raster plots, firing rate analyses, and network metrics for studying neural coding and dynamics.  
  - Prepared framework for integration with reinforcement learning agents.  
- [Project folder](Spiking_NN_Simulation/)
