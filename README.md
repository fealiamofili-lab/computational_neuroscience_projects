# Computational Neuroscience Projects

This repository contains in-progress projects focused on computational neuroscience, neural decoding, and biologically inspired learning. Each project is structured with READMEs, placeholder scripts, and planned pipelines to demonstrate methodology and active development.

## Projects

1. **BCI Virtual Arm**  
   - Goal: Decode motor imagery EEG signals to control a virtual arm in real-time.  
   - Status: In Progress  
   - [Folder link](BCI_Virtual_Arm/)

2. **Spiking Neural Network Simulation**  
   - Goal: Model small cortical microcircuits using excitatory/inhibitory neurons and STDP.  
   - Status: In Progress  
   - [Folder link](Spiking_NN_Simulation/)

3. **Alzheimer’s MRI Classification**  
   - Goal: Detect early-stage Alzheimer’s disease from MRI scans using CNNs.  
   - Status: In Progress  
   - [Folder link](Alzheimer_CNN/)

## Notes

- All projects are in-progress and demonstrate structured coding pipelines, methodology planning, and placeholder scripts.
- Dependencies and detailed planned pipelines are included in each project folder.
