# Spiking Neural Network Simulation of Cortical Microcircuits

**Goal:** Model cortical microcircuits with excitatory and inhibitory neurons and implement STDP learning rules.

**Status:** In Progress

**Skills Demonstrated:**  
- Spiking neural network modeling in Brian2  
- Implementation of spike-timing dependent plasticity (STDP)  
- Raster plots, firing rate analyses, and network metrics  
- Benchmarking network behavior against experimental neural data  
- Framework prepared for integration with reinforcement learning agents  

**Planned Pipeline:**  
1. Build excitatory/inhibitory network  
2. Apply STDP learning rules  
3. Simulate neural responses to input patterns  
4. Generate raster plots and analyze network metrics  
5. Explore integration with reinforcement learning agents for sensorimotor feedback  

**Dependencies:** Python, Brian2, NumPy, Matplotlib  

*This project reflects skills and methodology outlined in my CV: computational modeling, neural dynamics, and biologically inspired learning.*
