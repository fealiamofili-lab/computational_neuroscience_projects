# Spiking Neural Network Simulation of Cortical Microcircuits

**Goal:** Model cortical microcircuits with excitatory/inhibitory neurons and STDP learning rules.

**Status:** In Progress

### Pipeline
1. Build cortical microcircuits in Brian2/NEST
2. Apply STDP learning rules
3. Run simulations with input patterns
4. Analyze raster plots, firing rates, network metrics
5. Prepare framework for reinforcement learning agents

### Dependencies
Python, Brian2, NumPy, Matplotlib
