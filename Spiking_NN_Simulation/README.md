# Spiking Neural Network Simulation

**Goal:** Model small cortical microcircuits using excitatory/inhibitory neurons and STDP.

**Status:** In Progress

**Planned Pipeline:**
1. Build excitatory/inhibitory spiking network
2. Implement STDP learning
3. Run simulations
4. Analyze firing patterns and network encoding

**Dependencies:** Python, Brian2, NumPy, Matplotlib
